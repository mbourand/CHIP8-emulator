### Installation
Choisissez la version correspondant à votre système dans les releases.

### Compilation

#### Linux
```g++ -Iinc -O3 src/* -o emu -lsfml-system -lsfml-graphics -lsfml-window -lsfml-audio```

### Utilisation
```./emu <chemin_rom>```
